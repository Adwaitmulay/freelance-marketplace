# freelance-marketplace


A freelance service marketplace website built using HTML, CSS and JavaScript.

## About

A platform connecting freelancers and clients for short-term work. Freelancers can create profiles, list services, and clients can browse, message, and leave reviews.

## Team

- Adwait Mulay
- Ghratachi Rathore

## Features

- Signup as Freelancer or Client
- Login system
- Post a gig with title, category, description and price
- Browse all gigs with search and category filter
- Freelancer profile page with bio, gigs, and reviews
- Star rating and review system
- In-app messaging between client and freelancer
- All data saved using localStorage

## Tech Used

- HTML
- CSS
- JavaScript
